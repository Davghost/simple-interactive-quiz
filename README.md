# Quiz Game in C++

This project is a simple command-line quiz game written in C++.

The program reads questions from a text file, asks them to the user, counts the correct answers, and saves the player score in a ranking file.

## Features

- Multiple-choice questions (A–D)
- Answer validation
- Score calculation
- Ranking saved in a text file
- Ranking sorted by highest score

## Files

- `quizcode.cpp` – Main program file
- `perguntas.txt` – Questions file
- `ranking.txt` – Players ranking file
